# House BZ - Luxury Rental Platform

House BZ is a 100% crypto-native luxury rental platform that redefines exclusivity through decentralization.

## 🚀 Overview

This platform allows users to book extraordinary properties (Villas, Mansions, Ski Resorts) using cryptocurrencies (BTC, ETH, SOL, USDC). It combines blockchain security with high-end concierge services.

## 📖 Documentation

For detailed information about the project, please refer to:
- [MANUAL.md](./MANUAL.md) - Project conception, business model, and technical architecture.

## 🛠️ Tech Stack

- **Frontend**: React 19, Vite, Tailwind CSS, Motion.
- **Backend**: Firebase (Auth & Firestore).
- **Web3**: RainbowKit, Wagmi, Viem.
- **i18n**: i18next (English, Portuguese, Spanish, and more).

## 📦 Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`.
3. Set up environment variables (see `.env.example`).
4. Run the development server: `npm run dev`.
